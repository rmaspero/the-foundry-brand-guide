# Layouts & Floor Plans

Drop floor plans / layout drawings in here, then list each one inside
`<template id="gallery-floorplans">` in `index.html`.

Example:

```html
<template id="gallery-floorplans">
  <img src="floorplans/ground-floor.png" alt="Ground floor — overview" />
  <img src="floorplans/first-floor.png"  alt="First floor — overview" />
</template>
```
